Recovery BI Dashboard

About the Project

The Recovery BI Dashboard project provides a comprehensive solution for tracking brokerage receivables. It allows users to monitor pending amounts, due dates, transaction details, and payment statuses for clients across multiple projects. The project ensures better financial tracking and improved cash flow management.

Key Features

  • Detailed SQL views containing key data points like client name, project ID, brokerage value, transaction dates, and statuses.
  • Automated daily updates through stored procedures to ensure the latest data is available.
  • Dynamic Power BI dashboards with filters for status, client name, project name, and more.
  • Calculation of KPIs such as the rate of return on brokerage value, average days for payment, and transaction patterns.

Challenges

One of the primary challenges was handling multiple receivable dates for the same client and project ID, due to delays in payments. To overcome this, the latest receivable date for each client and project ID was mapped accurately, ensuring data consistency and reliability.

Results and Impact

Results

Increased efficiency: Payment tracking improved by 40% due to streamlined data processing and dashboard insights.

Data accuracy: Enhanced to 98%, ensuring reliable financial reporting and decision-making.

Business impact: Accelerated cash recovery timelines, reducing pending payment periods by an average of 20%.

Technology Stack

  • Database: SQL Server (Views, Stored Procedures)
  • Visualization: Power BI (Dynamic KPIs and interactive filters) & DAX
  • Programming: Advanced SQL Functions * Queries

The Recovery BI Dashboard has streamlined the monitoring of brokerage receivables at Kunvarji. By providing a clear overview of pending payments, transaction statuses, and due dates, it ensures that the finance team can proactively manage cash flows, identify payment delays, and enhance client follow-ups, leading to improved financial health and operational control.
