Centralized Client Database

About the Project

The Centralized Client Database project was developed to unify client data across multiple verticals at Kunvarji Group, including Real Estate Advisors, Wealth Solutions, Agro Solutions, and more. With a focus on streamlining operations, the project enabled the organization to track and manage client information in a single system, improving data visibility and creating opportunities for cross-selling between verticals.

This initiative began with extensive data normalization due to inconsistencies in client details spread across multiple servers and databases. After normalization, the data was consolidated into a unified SQL view, enabling accurate and actionable analysis for various stakeholders.

Key Features

  • Data consolidation across multiple business verticals.
  • SQL view development with conditional columns for client status tracking.
  • Automated stored procedures to refresh data daily for Power BI dashboards.
  • Advanced client mapping using soundex functions, achieving 97% data accuracy.
  • Enhanced cross-selling by identifying common clients across business units.

Challenges Faced

One of the significant challenges was identifying client existence across verticals due to inconsistent data formats and the absence of unique identifiers in some verticals. For instance, mobile numbers were often the only common data point, but duplicates and variations in names complicated the mapping process.

To address these issues, soundex functions and advanced SQL techniques were used to normalize data and achieve a mapping accuracy of 97%. This ensured reliable and actionable insights while minimizing errors.

Results and Impact

Key Achievements:

Data Accuracy: Achieved 97% accuracy in mapping client data across verticals.

Efficiency Gains: Improved data retrieval times by avoiding direct database querying.

Cross-Selling Impact: Enhanced cross-vertical business opportunities through consolidated client visibility.

Technology Stack

  • Database: SQL Server (Views, Stored Procedures)
  • Visualization: Power BI (Dynamic KPIs and interactive filters)
  • Programming: Advanced SQL Functions (e.g., soundex) & Queries

The Centralized Client Database has become a cornerstone for Kunvarji Group's data-driven decision-making, empowering the business to leverage insights effectively and improve operational efficiency across all verticals.
